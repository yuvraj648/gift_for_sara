# Sara Surprise ✨

A romantic interactive webpage for Sara.

## How to run this on your computer

1. **Extract the Zip file** to a folder.
2. **Install Node.js**: If you don't have it, download and install it from [nodejs.org](https://nodejs.org/).
3. **Open Terminal / Command Prompt** in the project folder (`sara-surprise`).
4. **Install dependencies**:
   ```bash
   npm install
   ```
5. **Start the surprise**:
   ```bash
   npm run dev
   ```
6. **Open the link**: The terminal will show a link like `http://localhost:5173`. Open it in your browser!
